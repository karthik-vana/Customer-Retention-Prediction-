Architecture

A deep dive into the machine learning pipeline powering our retention intelligence engine.

01
Data Ingestion & Cleaning Pandas

Raw telemetry and billing data is ingested, cleansed, and normalized. We handle missing values, outlier detection, and type casting to ensure high-quality input for the model.

7043 Training Samples
02
Feature Engineering NumPy

Transforming raw fields into predictive signals. We apply One-Hot Encoding for categorical variables like 'PaymentMethod' and 'Contract', and scale numerical features using RobustScaler to handle skewed distributions.

03
Model Training Scikit-Learn

We utilize an ensemble Random Forest Classifier tuned via GridSearchCV. The model is trained on balanced classes using SMOTE to prevent bias towards majority classes.

100 Decision Trees
Gini Impurity Criterion
04
Inference Engine Flask/Serverless

The model is serialized and deployed on a serverless Vercel runtime. It accepts JSON payloads, preprocesses them in real-time using the saved scaler artifacts, and returns sub-second predictions.

~120ms Avg Latency
99.9% Uptime
